# Crypto Price Predictor

This repository keeps the related crypto tools in separate product branches.

## Branches

| Branch | Product | Notes |
| --- | --- | --- |
| `main` | CryptoPredictor | Default branch. Streamlit predictor and Windows portable build. |
| `btc-forecast-live` | BTC Forecast Live | Root-domain forecast site served from `https://144-31-84-161.sslip.io/`. |
| `crypto-screener` | CryptoScreener | FastAPI futures screener, charts, alerts, and Windows portable build. |
| `oi-screener` | OI Screener | Streamlit open-interest screener and Windows portable build. |
| `timesfm-dashboard` | TimesFM dashboard | Experimental dashboard branch. |

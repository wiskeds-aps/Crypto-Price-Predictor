# BTC TimesFM forecast bot

Получает завершённые свечи BTCUSDT из публичного Binance API и строит
вероятностный прогноз TimesFM с XReg по рыночным факторам. Учитываются
доходности, моментум, волатильность, диапазон свечи, объёмы, число сделок и доля
агрессивных покупок. Торговые операции бот не выполняет, API-ключи не требуются.

## Запуск

```bash
cd /root/timesfm-project
source /root/timesfm-env/bin/activate
python bot_1h.py
```

По умолчанию используются 512 часовых свечей и прогноз на следующие 24 свечи.
Первый запуск загружает веса модели с Hugging Face и занимает больше времени.

Примеры:

```bash
# 15-минутные свечи, прогноз на 4 часа
python bot.py --interval 15m --context 512 --horizon 16

# Повторять прогноз каждый час
python bot_1h.py --repeat 3600
```

Результаты записываются в `forecast.json` и `forecast.csv`. Прогноз модели
экспериментальный и не является финансовой рекомендацией.

## Пятиминутный прогноз на 24 часа

```bash
python bot_5m.py
```

Используются 999 завершённых пятиминутных свечей. Результат содержит 288 точек
в `forecast_5m_24h.csv`: первая строка — следующие 5 минут, последняя — прогноз
через 24 часа.

При каждом запуске отдельные копии CSV и JSON сохраняются в каталогах
`history/5m` и `history/1h` с UTC-временем запуска в имени файла. Основные
файлы прогноза при этом перезаписываются актуальными данными.

Пример: `forecast_BTCUSDT_5min_2026-06-21_17-06-05_UTC.csv`.

## Структура

- `market_data.py` — загрузка и подготовка свечей Binance;
- `timesfm_model.py` — загрузка и прогноз TimesFM;
- `factor_model.py` — признаки и подготовка ковариат для XReg;
- `bot.py` — общая логика объединения и сохранения результатов;
- `bot_1h.py` — профиль запуска на часовых свечах;
- `bot_5m.py` — профиль запуска на 5-минутных свечах.
